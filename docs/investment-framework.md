# 投资与策略分析框架

本文件保存详细投资分析框架。`AGENTS.md` 只保留触发规则，避免编码任务反复消耗上下文。

## 总原则

当分析股票、基金、ETF、指数基金、AI产业、半导体、CPO、存储、算力、电力、纳斯达克、美联储、宏观流动性、A股、港股、美股、QDII、定投、一次性买入、止盈、资产配置或产品内策略推荐逻辑时，使用：

**周期 + 产业 + 流动性 + 估值 + 指数基金方法论 + 风险控制 + 操作策略**

所有建议仅作为投资分析和产品策略参考，不构成具体投资顾问建议，不承诺收益。

## 1. 周期位置

结合周金涛周期理论和市场实际，判断当前处于：

- 康波周期、产能周期、库存周期、资本开支周期、美元周期、债务周期、技术革命周期中的哪类阶段。
- 复苏初期、繁荣中段、过热尾部、衰退前期、出清底部，还是新一轮技术周期启动。
- 标的是产业趋势刚启动、主升浪中段、扩散补涨、订单验证、业绩兑现、扩产兑现、估值透支、周期尾声，还是产能过剩风险期。

重点看：技术革命是否真实发生，资本开支是否扩张，企业盈利是否兑现，市场是否透支未来预期，资金抱团是否过热。

## 2. AI基础设施链条

把 AI 理解为长期基础设施周期，类似电力、铁路、互联网、云计算。

产业链拆解：

- 上游：GPU、ASIC、HBM、DRAM、NAND、先进封装、半导体设备、材料、EDA、晶圆制造。
- 中游：AI服务器、数据中心、光模块、CPO、液冷、PCB、连接器、电力设备、储能、UPS、变压器。
- 下游：云厂商、AI应用、企业软件、机器人、自动驾驶、端侧AI、AI手机、AI PC。

分析时回答：

- 谁是真正受益方？
- 谁已经被充分定价？
- 谁还处于早期扩散？
- 谁的业绩弹性更大？
- 谁只是概念炒作？
- 谁可能面临扩产后的价格战？

阶段判断：预期驱动、订单验证、业绩兑现、扩产竞争、供需紧张、供需平衡、价格战、产能过剩、估值回归。

## 3. 宏观与流动性

必须考虑：

- 美联储利率路径、加息/降息预期、美债收益率、美元指数。
- 通胀、就业、全球流动性、人民币汇率。
- A股、港股、美股资金流向。
- QDII额度、限购、溢价和汇率风险。

高利率持续时，提示高估值成长股、纳斯达克、半导体、AI主题、长久期资产的估值压力。

降息周期要区分：

- 预防式降息：通常利好风险资产和成长股估值。
- 衰退式降息：短期可能先杀盈利，再修复估值。

## 4. 估值与情绪

必须区分：

- 好公司、好产业、好指数、好基金、好价格。

估值参考：

- PE、PB、股息率、盈利收益率、ROE、PEG、自由现金流、历史估值分位、盈利增速、市场情绪。

涨幅较大板块要检查：

- 业绩兑现不及预期。
- 订单高峰后回落。
- 扩产导致供需恶化。
- 估值过高。
- 资金抱团瓦解。
- 宏观流动性收紧。
- 主题炒作退潮。

下跌板块要区分：

- 周期底部机会。
- 估值杀到合理区。
- 产业逻辑被破坏。
- 基本面恶化。
- 长期竞争力下降。

## 5. 指数基金方法论

当涉及指数基金、ETF、宽基基金、行业基金、QDII、定投、长期配置、普通人理财时，参考银行螺丝钉《指数基金投资指南》的核心思想：

- 普通投资者优先考虑低成本、分散化、长期定投的指数基金。
- 宽基通常更适合作为底仓，行业/主题指数更适合作为卫星仓。
- 行业主题需要更严格仓位控制。

盈利收益率法：

```text
盈利收益率 = 1 / 市盈率
```

- 盈利收益率高，指数相对便宜。
- 可与长期国债收益率、货币基金收益率、无风险收益率比较。
- 适合盈利稳定的宽基、银行、红利、消费等，不适合亏损行业或盈利剧烈波动行业。

博格公式：

```text
长期收益 ≈ 初始股息率 + 盈利增长率 + 估值变化
```

需要判断收益来源主要来自盈利增长、股息，还是估值扩张。

估值区间操作：

- 低估区：可开始定投、分批买入、提高定投，但不能满仓。
- 合理区：可继续持有、正常定投，不宜追高。
- 高估区：谨慎新增、降低定投、分批止盈。
- 泡沫区：停止追高、分批止盈、降低主题仓位。

定投不是无脑买入。必须判断标的是宽基、行业、主题还是个股；估值是否合理；是否具备长期生命力；用户是否能承受回撤。

## 6. 中美市场差异

美国市场：

- 长期由盈利增长和科技创新驱动。
- 标普500偏核心资产，纳斯达克100偏科技成长和长久期资产。
- 美债收益率对估值影响明显。
- 龙头集中度高，美元周期影响全球资金流向。

中国市场：

- 估值波动和情绪周期明显。
- 政策、流动性、产业趋势影响更强。
- 宽基低估不代表马上上涨。
- 行业主题容易快速主升，也容易快速回撤。

策略区别：

- 美国指数基金更适合长期定投和全球核心配置，但要注意高估值、利率、汇率、QDII溢价。
- 中国指数基金更适合低估分批布局、增强定投、高估分批止盈。

## 7. 仓位管理

核心仓、卫星仓、观察仓不由行业名称决定，而由资产分散度、盈利稳定性、长期生命力、估值、周期、波动率、回撤风险、流动性、用户风险承受能力决定。

仓位表达建议：

- 观察仓：0%-5%。
- 轻仓：5%-10%。
- 标准仓：10%-20%。
- 偏重仓：20%-30%。
- 重仓：30%以上，需要极强确定性，普通投资者谨慎使用。

高波动、高估值、高主题化资产，即使长期逻辑成立，也不宜轻易重仓。

## 8. 输出结构

投资分析优先按以下结构输出：

1. 当前结论：早期、中期、后期、过热、回调机会，还是风险释放中。
2. 周期位置：结合产业、库存、资本开支、流动性、估值周期。
3. 产业逻辑：产业链位置、受益逻辑、当前阶段。
4. 指数基金视角：宽基/行业/主题、核心仓/卫星仓、估值、定投适配性、QDII风险。
5. 宏观与美联储风险：利率、美元、美债、通胀、流动性、汇率。
6. 估值与情绪：低估、合理、高估、泡沫化、贵但有业绩支撑，还是便宜但逻辑变差。
7. 操作建议：一次性买入比例、定投节奏、回调加仓、止盈/减仓、最大仓位、不适合操作的情况。
8. 最大风险：列出最重要的 3 个具体风险。

回答风格：像资深投资经理，不像营销号；不只讲故事，要落到周期、估值、仓位、风险和操作。
